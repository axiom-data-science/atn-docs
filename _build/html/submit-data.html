How to Submit Deployment Metadata¶

Depending on your project funding and other factors, you may need to submit pre-deployment metadata and post-deployment metadata for your project. To do this, use the ATN Tag Deployment Metadata Template as described in the steps below.

Note

You will need a Research Workspace account to submit metadata. If you haven’t created one yet, please see the Creating & Accessing an Account tutorial.

Pre-Deployment Metadata¶

  1. Log in to your Research Workspace account.
  2. Navigate to your project.
  3. Click on the Metadata template folder in the folder tree on the left.
  4. Download either the XLSX or CSV version of the ATN Tag Deployment Metadata Template file (whichever format your prefer to work with; both files are the same).
  5. Fill in the first 5 columns (Columns A-F, which have a green background in the XLSX file) with information for each tag (1 row per tag).
  6. Save the file somewhere easy to find on your computer.
  7. Upload the completed file to the Data folder of your project in the Research Workspace.

Post-Deployment Metadata¶

  1. Download the pre-deployment metadata file you uploaded in the steps above.
  2. Fill in the remaining columns (F-P, shaded blue in the XLSX file) with deployment information (1 row per tag).
  3. Save the file somewhere easy to find on your computer.
  4. Upload the completed file to the Data folder of your project in the Research Workspace.

How to Submit Deployment Data¶

As of July 2018, data can be submitted to the IOOS ATN by uploading it to the Research Workspace, a web-based data management platform developed to facilitate the management, analysis, and publication of scientific data, by following these steps:

  1. Create a Research Workspace account
  2. Navigate to your project
  3. Upload your data files

For assistance submitting data to the IOOS ATN, see the Research Workspace help docs, consult the tutorials listed below, or contact Axiom Data Science using the information on the Contact Us page.
